# My Vim Config.

### Description
Presenting my .vimrc file. To be used as an educational tool for others (i.e actually try to understand what's going on) and a reference for me when setting up machines outside of my own. 

### Installation:
User must install the Vim plugin manager, Vundle:
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

```

Install the plugins in the vimrc file by running the following command in vim:
```
:PluginInstall
```

Restart Vim and changes should be effect.
